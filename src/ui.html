Welcome to Themer
Themer lets you quickly create color, text and effect themes from your published Figma styles.

Themer does this by matching styles with the same names across themes.
Make a JSON Bin account
Themer stores the names and unique identifiers of your styles outside Figma so that you can use them across team members and files. To get started you'll need a free account from jsonbin.io.
Generate an API key
After signing up, you'll need to generate an API key. You can find this by clicking on your user avatar and selecting "API Keys".
Add your credentials
Add your API key to the Admin tab of Themer. A URL to your bin will be auto-generated when you save your settings.
Start creating themes
Create themes by running Themer inside your libraries. Styles must be published tin order for Themer to work.
